This directory contains one page for each Software Carpentry lesson package. Each of those pages uses {% raw %}{% include %}{% endraw %} to include all of the lesson material for viewing and checking.

Material from The Hacker Within at the University of Wisconsin - Madison

The Unix Shell
This introduction is based on materials by Milad Fatenejad, Sasha Wood, and Radhika Khetani, and includes a single tutorial.
Version Control With Git
This introduction is based on materials by Katy Huff and Anthony Scopatz as modified by Joshua R. Smith and Sri Hari Krishna Narayanan. It includes:
  • Instructors' Guide
  • Local Operations
  • Using Remote Repositories
Introduction to Python
An introduction to the basic features of Python, which covers:
  • Variables and Types
  • Flow Control
  • Data Structures
  • Strings and File I/O
Most sections also include some exercise material.
Python Debugging
This introduction is based on materials originally written by Anthony Scopatz and Patrick Fuller. It includes a single tutorial.
NumPy
An introduction to array-based computing in Python from Matthew Terry. It consists of a single tutorial.
SciPy
A supplement to the NumPy tutorial from Anthony Scopatz. It also consists of a single tutorial.
Plotting in Python with matplotlib
An introduction to plotting in Python using matplotlib originally written by Anthony Scopatz and Katy Huff. It consists of a single tutorial.
Documentation
An introduction to documenting code from Anthony Scopatz. It consists of a single tutorial.

Instructor's Guide

Instructor's Guide for the Unix Shell
Core material for the Unix shell: the organization of files and directories, creating and deleting things, the pipe and filter model, processing multiple files using loops, putting commonly-used commands in shell scripts, and finding things.
To do: diagrams.
Instructor's Guide for Basic Python Using IPythonBlocks
An introduction to basic programming concepts using the IPython Notebook and IPythonBlocks.
To do: lots.
Instructor's Guide for Sets and Dictionaries
Optional material for Python programming: sets and dictionaries, how hash-based data structures work, the JSON data format (which relies heavily on dictionaries), and three examples of increasing complexity.
To do: diagrams.
Instructor's Guide for Invasion Percolation Example
Invasion percolation is a good half-day example that brings together many of the things we teach about programming while introducing new concepts like performance profiling and development lifecycles.
To do: diagrams; challenges; IPython Notebooks.
Instructor's Guide for Databases and SQL
Core material for SQL databases: selection, removing duplicates, filtering with Boolean conditions, performing calculations on values, sorting, handling NULLs, aggregating values, grouping, joining multiple tables, a very brief introduction to creating tables and inserting data, an equally brief coverage of transactions, and a couple of examples showing how to uses databases from inside a program.
To do: diagrams, and material on how to design and structure tables.
Instructor's Guide for Web Programming with Python
Preliminary material on web programming: a little bit of history, basic HTML, using a templating engine to create pages, how HTTP works, fetching data programmatically, providing data by dynamically generating static pages, why and how to create an index, and data syndication.
To do: diagrams; switch from Jinja2 to Jekyll (since that's what GitHub uses); try out.
