Do you spend too much time wrestling with computers,
and not enough doing research?
We can help.

Our mission is to teach researchers basic lab skills for scientific computing so that they can get more done in less time and with less pain. In the past year alone, we have run almost one hundred two-day workshops for more than 3500 scientists on:

The Unix Shell for automating repetitive tasks

Git and GitHub for tracking and sharing work

Python or R for building programs reliably

Databases, SQL, and data management

All our lessons are freely available under the Creative Commons - Attribution license, and all our instructors are volunteers, so the only cost to host a workshop is travel and accommodation (though donations to help cover administration, instructor training, and curriculum development are always welcome).
