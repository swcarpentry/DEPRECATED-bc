  • Code on the left will be displayed just as it's written within the xmp tags.
  • Unfortunately, the code cannot be styled to wrap within the xmp element.
  • To get text to wrap in the code display, you could use <pre> tags instead of <xmp> tags, but these would require you to type out "& l t" and "& g t" instead of the "<" and ">" symbols in the code.
