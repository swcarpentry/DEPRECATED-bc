Python is...

a popular and easy to learn scripting language

a tool to generate figures

a readable, whitespace delimited, programming language

Free!

So for those of you who have never encountered Python before, what is it? Python is a programming language which is used to: - develop analysis pipelines - process and analyse data - visualise data, and generate publication quality graphics - develop new statistical methods It is also a free open-source project, making it completely free to download and extend.

Tasks that Python could help you do:

Transform your data files from one arbitraty format to another.

Analyze image files.

Loop through all the files in a driectory, perform an analysis on them, and plot the results.

Almost anything!

Why Python and not <Perl> (or something else)?

It is easy to learn, and easy to use!

Popularity matters! If it is what your collaborators are using it, may be a good idea for you to learn it too.

People are actively developing packages and tools.

Python has been widely adopted and is continuing to gain in popularity.
